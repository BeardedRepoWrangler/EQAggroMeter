# Log

Dated work log. One file per day of real work, summarizing what changed and why.

## Rules

- **Filename pattern**: `YYYY-MM-DD.md`
- One file per working day; append throughout the day if needed.
- Aim for "what changed and why," not "what I touched." Future-you should be able to skim and reconstruct.

Use [[templates/log-entry]] when starting a new one.
