# Runbooks

How-to guides for tasks that recur or are easy to forget — setup, deploys, debugging walkthroughs, recovery procedures.

## What belongs here

- Step-by-step procedures with commands
- Common diagnostics and fixes
- Anything you'd want to follow without re-deriving

Use [[templates/runbook]] when starting a new one.
